Checklist тестирование программы, которая определяет тип треугольника по трем его сторонам. Каждая из сторон задается в отдельном текстовом поле.

1. Ввести значения для равностороннего треугольника. (6, 6, 6)
2. Ввести значения для равнобедренного треугольника. (3, 3, 5)
3. Ввести значения для остроугольного треугольника. (66, 67,68)
4. Ввести значения для прямоугольного треугольника. (3, 4, 5)
5. Ввести значения для тупоугольного треугольника. (2, 3, 4)
6. Ввести десятичное число (1.24, 23.4, 5)
7. Ввести отрицательные числа. (-2, -13, -14)
8. Оставить поля пустыми.
9. Ввести числа буквами.
10. Ввести буквы. (a, c, e)
11. Ввести значения не существуюшего треугольника (2, 10, 3)
12. Ввести очень большие числа.
